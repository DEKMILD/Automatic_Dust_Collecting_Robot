โครงการ "หุ่นยนต์เก็บฝุ่นอัตโนมัติ" เป็นโครงการที่มุ่งเน้นการพัฒนาอุปกรณ์อัจฉริยะสำหรับทำความ
สะอาดพื้นโดยอัตโนมัติ โดยอาศัยเทคโนโลยี Internet of Things (IoT) และระบบคอมพิวเตอร์ฝังตัว 
(Embedded Systems) เพื่อเพิ่มประสิทธิภาพในการทำความสะอาดและลดภาระของผู้ใช้งาน หุ่นยนต์นี้ถูก
ออกแบบมาให้สามารถเคลื่อนที่ได้เอง ปรับเปลี่ยนทิศทางตามสภาพแวดล้อม และเชื่อมต่อกับระบบควบคุมผ่าน
อินเทอร์เน็ต ทำให้สามารถใช้งานได้ง่ายและสะดวกมากขึ้น
